# 037 View

[![037 View](https://img.youtube.com/vi/DwUFTBblMWM/0.jpg)](https://www.youtube.com/watch?v=DwUFTBblMWM)

The SQL in the `exercise1.sql` file
- Should create a view titled `Employee_Contact` with the following columns
    - The current date titled as "Current Date"
    - Employee's `FIRST_NAME`
    - Employee's `LAST_NAME`
    - Employee's `EMAIL`
    - Employee's `PHONE_NUMBER`
    - Employee's `ADDRESS`
    - Employee's `ZIP_CODE`
- Should gather the appropriate columns from the `Employee` and `Contact_Info` tables, ONLY for employee's existing in both tables

The SQL in the `exercise2.sql` file
- Should select all rows and columns from the view you created in `exercise1.sql`
